int dfk_tcp_socket_init ( dfk_tcp_socket_t *  sock,
dfk_t *  dfk 
)

Initialize TCP socket object

Returns
  • dfk_err_ok On success
  • dfk_err_sys If socket(2) syscall failed

TCP socket object is manually initialized bypassing dfk_tcp_socket_init method when a new incoming connection is accepted. When adding any new code to the dfk_tcp_socket_init method and/or new members to the dfk_tcp_socket_t structure don't forget to handle them in the dfk_tcp_socket_listen.
