Error codes returned by dfk functions

Enumerator
dfk_err_ok 

No error

dfk_err_eof 

End of file (stream, iterator, etc)

dfk_err_busy 

Resource is already acquired.

dfk_err_nomem 

Memory allocation function returned NULL

dfk_err_notfound 

Object not found

dfk_err_badarg 

Bad argument

Passing NULL pointer to the function that expects non-NULL will yield this error.

dfk_err_sys 

System error, see dfk_t.sys_errno

dfk_err_inprog 

The operation is already in progress

dfk_err_panic 

Unexpected behaviour, e.g. unreachable code executed

Please submit a bug report if API returns this error code. https://github.com/ivochkin/dfk/issues/new

dfk_err_not_implemented 

Functionality is not implemented yet

dfk_err_overflow 

Floating point, or integer overflow error.

dfk_err_protocol 

Protocol violation

For instance, invalid HTTP request method.

dfk_err_timeout 

Timeout has expired

dfk_err_connection_refused 

Connection refused

Method dfk_stop() have been called
